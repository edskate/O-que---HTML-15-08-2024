O que é HTML.

HTML é uma sigla para HyperText Markup Language.

E uma linguagem de marcação utilizada para dizer ao seu navegador como struturar a página.

Elementos:

O Fechamento das tags podem transformar uma parte do contéudo dentro do elemento em um link para direcionar á uma outra página web, colocar a palavras em itálico, e assim por diante.

Aninhamento elementos.

Elementos podem ser inseridos dentro de outros elementos- isto é chamado de aninhamento.

Exemplo:Como destacar uma frase:

Meu gato é muito mal-humorado.
